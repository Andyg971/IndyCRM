# IndyCRM
# 🏢 IndyCRM

## 📋 Description
IndyCRM est une application de gestion tout-en-un conçue spécifiquement pour les indépendants et les TPE. Elle simplifie la gestion quotidienne de votre activité en regroupant tous les outils essentiels dans une seule interface intuitive.

## ✨ Fonctionnalités Principales

### 👥 Gestion des Clients
- Base de données clients complète
- Historique des interactions
- Gestion des coordonnées et informations importantes
- Notes et commentaires personnalisés
- Suivi des préférences clients

### 💰 Facturation
- Création de factures professionnelles
- Suivi des paiements
- Gestion des devis
- Rappels automatiques
- Export PDF et formats multiples
- Numérotation automatique des factures

### 📊 Suivi des Projets
- Tableau de bord des projets en cours
- Gestion des délais et échéances
- Suivi de l'avancement
- Attribution des tâches
- Notes et documents associés

### 📈 Tableaux de Bord et Rapports
- Vue d'ensemble de l'activité
- Statistiques et graphiques
- Rapports personnalisables
- Indicateurs de performance
- Prévisions financières

## 🛠 Spécifications Techniques
- Application native macOS/iOS
- Développée avec Swift et SwiftUI
- Base de données Core Data
- Synchronisation iCloud (optionnelle)
- Interface moderne et responsive

## 🔒 Sécurité et Confidentialité
- Données stockées localement
- Chiffrement des données sensibles
- Sauvegardes automatiques
- Conformité RGPD

## 🎯 Public Cible
- Indépendants
- Auto-entrepreneurs
- TPE/PME
- Consultants
- Freelances
- Artisans

## 🚀 Installation
1. Téléchargez l'application depuis l'App Store
2. Lancez IndyCRM
3. Suivez l'assistant de configuration
4. Commencez à gérer votre activité !

## 💻 Configuration Requise
- macOS 12.0+ ou iOS 15.0+
- 4 Go de RAM minimum
- 500 Mo d'espace disque
- Connexion Internet pour la synchronisation (optionnelle)
