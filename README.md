# IndyCRM

## Description
IndyCRM est un logiciel de gestion de clients, de facturation et de suivi des projets, conçu spécialement pour les indépendants et TPE.

## Fonctionnalités principales
- **Gestion des clients** : Stockez et gérez les informations de vos clients
- **Facturation** : Créez, envoyez et suivez vos factures
- **Suivi de projets** : Gérez vos projets, suivez leur avancement
- **Tableaux de bord** : Visualisez vos données importantes en un coup d'œil
- **Rapports** : Générez des rapports sur votre activité

## Architecture technique
Ce projet est développé en Swift pour macOS/iOS, utilisant:
- SwiftUI pour l'interface utilisateur
- Core Data pour la persistance des données
- CloudKit pour la synchronisation (optionnel)

## Installation
1. Clonez ce dépôt
2. Ouvrez le projet dans Xcode
3. Compilez et exécutez l'application

## Exigences système
- macOS 12.0+ ou iOS 15.0+
- Xcode 13.0+
- Swift 5.5+
